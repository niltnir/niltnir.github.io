Number of Non-Adjacent Subsets

How many subsets of \(\{1, 2, \dots, 100\}\) are there that do not contain adjacent numbers?

Solution

Let \(a_n\) be the number of subsets of \(\{1, 2, \dots, n\}\) that do not contain adjacent numbers. We claim that \(a_n = F_{n+2}\), where \(F_n\) denotes the \(n\)-th Fibonacci number.

First, let us compute the first two values of \((a_n)\). When \(n=1\), we have the subsets \(\{\emptyset, \{1\}\}\), so \(a_1=2\) and when \(n=2\), we have the subsets \(\{\emptyset, \{1\}, \{2\}\}\), so \(a_2=3\). It now suffices to prove that \(a_n = a_{n-1} + a_{n-2}\).

Consider the numbers \(1\), \(2\), \(\dots\), \(n\). We do casework on whether to include or exclude \(n\) in our subset.

Suppose we include \(n\). This means that we cannot include \(n - 1\). The exclusion of \(n-1\) does not affect the inclusion of the rest of the numbers between \(1\) and \(n-2\), so there will be \(a_{n-2}\) subsets that can be formed with these numbers.

On the other hand, if we exclude \(n\), then this does not affect the inclusion of the rest of the numbers from \(1\) to \(n-1\). Therefore, there will be \(a_{n-1}\) subsets that can be formed.

We have therefore shown that \(a_n = a_{n-1} + a_{n-2}\), so \(a_{n} = F_{n+2}\). Our particular problem asks for the case when \(n = 100\), so \[a_{100} = F_{102} = \boxed{927,372,692,193,078,999,176}.\]

\(\blacksquare\)
