Red-Dominated Rows and Blue-Dominated Columns

Let \(m\) and \(n\) be odd positive integers. Each square of an \(m\) by \(n\) board is coloured red or blue. A row is said to be red-dominated if there are more red squares than blue squares in the row. A column is said to be blue-dominated if there are more blue squares than red squares in the column. Determine the maximum possible value of the number of red-dominated rows plus the number of blue-dominated columns. Express your answer in terms of \(m\) and \(n\).

Solution

We solve a generalization of the problem for all positive integers \(m\) and \(n\).

Let \(r\) be the number of red-dominated rows, \(b\) be the number of blue-dominated columns, and \(M\) be the maximum possible value of \(r+b\). WLOG, if \(m\) is the width of the board, and \(m\geq n\), we claim that \[ M = m + \left\lfloor\frac{m\cdot\left\lfloor\frac{n - 1}{2}\right\rfloor}{\left\lfloor\frac{m}{2}\right\rfloor + 1}\right\rfloor,\] where \(\left\lfloor x\right\rfloor\) denotes the greatest positive integer less than or equal to \(x\). We do not have to worry about cases when \(m<n\), since by the symmetry fo the board, we can get an \(m\times n\) board from an \(n\times m\) board by rotating the board \(90^\circ\) and swapping the colors of all the squares.

We begin by showing that \(r + b \leq M\). Suppose FTSOC that \(r+b>M\). Let \(r+b=M+t\), where \(t\) is some positive integer. Note that since \(r\leq n\) and \(M+t-r=b\leq m\), we have \[ M+t-m \leq r \leq n.\] Now consider a coloring with \(r\) red-dominated rows and \(b=M+t-m\) blue-dominated columns. Since there are \(m\) squares in a row, at least \(r\cdot\left(\left\lfloor\frac{m}{2} \right\rfloor + 1\right)\) squares must be red. Similarly, at least \(b\cdot\left(\left\lfloor\frac{n} {2}\right\rfloor + 1\right)\) must be blue. However, notice that

\[\begin{align*} &\quad r\cdot\left(\left\lfloor\frac{m}{2}\right\rfloor + 1\right) + b\cdot\left(\left\lfloor\frac{n}{2}\right\rfloor + 1\right) \\ =&\quad r\cdot\left(\left\lfloor\frac{m}{2}\right\rfloor - \left\lfloor\frac{n}{2}\right\rfloor\right) + \left(r + b\right)\left(\left\lfloor\frac{n}{2}\right\rfloor + 1\right)\\ =&\quad r\cdot\left(\left\lfloor\frac{m}{2}\right\rfloor - \left\lfloor\frac{n}{2}\right\rfloor\right) + \left(M + t\right)\left(\left\lfloor\frac{n}{2}\right\rfloor + 1\right)\\ \geq&\quad\left(M+t-m\right)\left(\left\lfloor\frac{m}{2}\right\rfloor - \left\lfloor\frac{n}{2}\right\rfloor\right) + \left(M + t\right)\left(\left\lfloor\frac{n}{2}\right\rfloor + 1\right)\\ =&\quad\left(M+t\right)\left(\left\lfloor\frac{m}{2}\right\rfloor + 1\right) - m\cdot\left(\left\lfloor\frac{m}{2}\right\rfloor - \left\lfloor\frac{n}{2}\right\rfloor\right)\\ \geq&\quad\left(M+1\right)\left(\left\lfloor\frac{m}{2}\right\rfloor + 1\right) - m\cdot\left(\left\lfloor\frac{m}{2}\right\rfloor - \left\lfloor\frac{n}{2}\right\rfloor\right)\\ =&\quad\left(m+\left\lfloor\frac{m\cdot\left\lfloor\frac{n - 1}{2}\right\rfloor}{\left\lfloor\frac{m}{2}\right\rfloor + 1}\right\rfloor+1\right)\left(\left\lfloor\frac{m}{2}\right\rfloor + 1\right) - m\cdot\left(\left\lfloor\frac{m}{2}\right\rfloor - \left\lfloor\frac{n}{2}\right\rfloor\right)\\ >&\quad\left(m+\left\lfloor\frac{m\cdot\left\lfloor\frac{n - 1}{2}\right\rfloor}{\left\lfloor\frac{m}{2}\right\rfloor + 1}\right\rfloor\right)\left(\left\lfloor\frac{m}{2}\right\rfloor + 1\right) - m\cdot\left(\left\lfloor\frac{m}{2}\right\rfloor - \left\lfloor\frac{n}{2}\right\rfloor\right)\\ =&\quad m\cdot\left(\left\lfloor\frac{n - 1}{2}\right\rfloor + \left\lfloor\frac{n}{2}\right\rfloor + 1\right)\\ =&\quad mn, \end{align*}\]

where the equality \(\left\lfloor\frac{n - 1}{2}\right\rfloor + \left\lfloor\frac{n}{2}\right\rfloor + 1 = n\) can easily be derived by trivial casework on the parity of \(n\). This is a contradiction, as the sum of the minimum numbers of red and blue squares exceeds the total number of squares in the grid. Hence, \[r + b \leq M.\]

Lastly, we show that a coloring of the board exists such that \(r + b = M\). We color the board in steps. First, color \(\left\lfloor\frac{n}{2}\right\rfloor + 1\) of the rows in the grid blue and the remainder red. Then, note there are \(mn - m\cdot\left(\left\lfloor\frac{n}{2}\right\rfloor + 1\right) = m\cdot\left\lfloor\frac{n - 1}{2} \right\rfloor\) remaining red squares, some of which can be swapped with blue squares in other rows such that each pair being swapped remains within the same column. In particular, dividing the total number of red squares by the number of squares required to meet the quota of a red-dominated row, we see that we can have exactly \(\left\lfloor\frac{m\cdot\left\lfloor\frac{n - 1}{2}\right\rfloor}{\left\lfloor\frac{m}{2} \right\rfloor + 1}\right\rfloor\) red-dominated rows. Since swapping remains within columns, the number of blue-dominated columns remains invariant. It is now easy to see that a coloring in fact exists.

In conclusion, for an \(m\)-width, \(n\)-height board with \(m\geq n\), it is clear that \[ M = m + \left\lfloor\frac{m\cdot\left\lfloor\frac{n - 1}{2}\right\rfloor}{\left\lfloor\frac{m}{2}\right\rfloor + 1}\right\rfloor.\] And specifically for odd \(m\), \(n\), \[ M = m + \left\lfloor\frac{m\cdot\left(n - 1\right)}{m + 1}\right\rfloor,\] or simply \(M=m+n-2\) for \(m\geq n> 1\).
