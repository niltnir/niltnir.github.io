Relatively-Prime Reciprocals Sum to 1/2

For some positive integer \(n \geq 2\), suppose \(p\) and \(q\) are relatively prime positive integers such that \(p < q \leq n\) and \(p + q > n\). If \(\mathcal{R} _n\) is the set of all ordered pairs \((p, q)\), prove that \[\sum_{(p,q) \in \mathcal{R}_n} \frac{1}{pq} = \frac{1}{2}.\]

Solution

Given a set of positive integer pairs \(X = \{\left(a_1, b_1\right), \left(a_2, b_2\right), \dots, \left(a_n, b_n\right)\}\), define the function \(\sigma:X\to \mathbb{Q}\) as \[ \sigma(X) \coloneqq \sum_{(a,b) \in X}\frac{1}{ab}.\] Moreover, define the sets \(A_n \coloneqq \{(p,q)\in \mathcal{R}_n \mid p+q>n+1\} \), \(B_n \coloneqq \{(p,q) \in \mathcal{R}_n \mid p+q=n+1\}\), and \(N_n \coloneqq \{(p, n) \in \mathcal{R}_n\} \). We derive a few properties of these sets.

By construction, \(A_n\) and \(B_n\) are disjoint sets such that \(A_n \cup B_n = \mathcal{R}_n\) (ie. \(A_n = \mathcal{R}_n \setminus B_n\)). Furthermore, since the sum of each pair in \(A_n\) is greater than \(n+1\) and follow the conditions satisfied by \(\mathcal{R}_n\), the pairs appear in the set \(\mathcal{R}_{n+1}\). In other words, \(A_n \subset \mathcal{R}_{n+1}\).

\(N_n\) is also the complement of \(A_{n-1}\) with respect to \(\mathcal{R}_{n}\). \(A_{n-1} \subset \mathcal{R}_n\) from before, and each element of \(A_{n-1}\) is a pair whose greatest element is at most \(n-1\). Likewise, \(N_n \subset \mathcal{R} _n\), but each element of \(N_n\) is a pair whose greatest element is exectly \(n\). These sets therefore disjointly make up \(\mathcal{R}_n\).

We have shown that \(A_n \sqcup B_n = \mathcal{R}_n = A_{n-1} \sqcup N_n\), so it is clear that \[\sigma(A_n) + \sigma(B_n) = \sigma(\mathcal{R}_n) = \sigma(A_{n-1}) + \sigma(N_n)\] for each \(n > 2\).

We then proceed to prove by induction that \(\sigma(\mathcal{R}_n) = \frac{1} {2} \) for each \(n > 2\) (\(n=2\) is trivially true). The base case is \(n=3\), for which \(\mathcal{R}_3 = \{(1,3), (2,3)\}\), and \(\sigma(\mathcal{R} _3) = \frac{1} {3} + \frac{1}{6} = \frac{1}{2}\). Now suppose that for some integer \(k > 2\), \(\sigma(\mathcal{R}_k)=\frac{1}{2}\). By the equalities derived above, we have \[\begin{align*} \sigma(\mathcal{R}_{k+1}) &= \sigma(A_k) + \sigma(N_{k+1}) \\ &= \sigma(\mathcal{R}_k) - \sigma(B_k) + \sigma(N_{k+1}) \\ &= \frac{1}{2} + \sigma(N_{k+1}) - \sigma(B_k). \end{align*}\] Thus, it suffices to show that \(\sigma(N_{k+1}) = \sigma(B_k)\) for each \(k>2\).

We now prove a short lemma. For positive integers \(a\), \(b\), and \(m\) such that \(a+b=m\), \(a\perp m \iff a\perp b \iff b\perp m\). By symmetry, it suffices to prove that \(a\perp m\) if and only if \(a\perp b\). The Euclidean algorithm yields \(\gcd(m, a) = \gcd(a, m - a = b)\), so if one of them is \(1\) then so is the other.

Let us now compute \(\sigma(N_{k+1})\). We have \[\begin{align*} \sigma(N_{k+1}) &= \sum_{\substack{0<p<k+1\\ p\perp k+1}} \frac{1}{p(k+1)} \\ &= \frac{1}{2}\cdot\sum_{\substack{0<p<k+1\\ p\perp k+1}}\left(\frac{1}{p(k+1)}+\frac{1}{(k+1-p)(k+1)}\right) \\ &= \frac{1}{2}\cdot\sum_{\substack{0<p<k+1\\ p\perp k+1}}\frac{1}{p(k+1-p)} \\ &= \sum_{\substack{0<p<q\leq k\\ p+q=k+1\\ p\perp q}}\frac{1}{pq} \\ &= \sum_{\substack{(p,q)\in \mathcal{R}_k\\ p+q=k+1}}\frac{1}{pq} = \sigma(B_k), \end{align*}\] where our lemma is used on the second and the fourth equalities.

We have shown that \(\sigma(N_{k+1})=\sigma(B_k)\) for each \(k>2\), completing the proof.

\(\blacksquare\)

Motivation

The main difficulty of this problem is to get the whole induction to go through. We want to somehow link the \(n=k\) case to the \(n=k+1\) case, but simple computations of the sum do not yield anything fruitful. This motivates us to analyze the actual pairs instead. We see that some of the pairs appear multiple times across \(n\), so we question whether we have anything useful to say about them. One of the key realizations is to notice that if pair \((p, q) \in \mathcal{R}_k\) has the property \(p+q>k+1\), then that pair carries over to the next sum for \(n=k+1\). It is then easy to realize that the rest of the pairs that make up \(\mathcal{R}_{k+1}\) have largest element \(k+1\). This motivates the three sets we defined in the solution.

Beyond this, the rest of the problem simplifies to showing that the sums from these sets are equal. The insight here is that numbers relatively prime to some integer \(k\) come in pairs that sum to \(k\). This is the lemma we proved in the solution.
