Mysterious Sequence in N

Let \(n > 1\) be an integer. Find, with proof, all sequences \(x_1, x_2, \dots, x_{n-1}\) of positive integers with the following three properties: (a) \(x_1 < x_2 < \dots < x_{n-1}\); (b) \(x_i + x_{n-i} = 2n\) for all \(i = 1, 2, \dots, n-1\); (c) given any two indices \(i\) and \(j\) (not necessarily distinct) for which \(x_i + x_j < 2n\), there is an index \(k\) such that \(x_i + x_j = x_k\).

Solution

We claim that the answer is only the sequence \(2, 4, 6, \dots, 2n - 2\). It is easy to verify that all of the conditions are met. (a) and (b) are trivial. To verify (c), note that if we divide each of the terms by \(2\), the resulting sequence is simply the sequence of integers from \(1\) to \(n-1\). Since \(x_i + x_j < 2n\) is equivalent to \(\frac{x_i}{2} + \frac{x_j}{2} < n\), we know that there exists some index \(k\) for which \(\frac{x_i}{2} + \frac{x_j}{2} = \frac{x_k}{2}\), where \(\frac{x_k}{2}\) is one of the integers between \(1\) and \(n-1\).

We then show that this is the only sequence that works. We claim that \[\begin{align*} x_1 + x_{n-2} &= x_{n-1} \\ x_1 + x_{n-3} &= x_{n-2} \\ &\vdots \\ x_1 + x_2 &= x_3. \end{align*}\]

To begin, note that \[x_1 < x_2 \iff x_1 + 2n - x_2 < 2n \iff x_1 + x_{n-2} < 2n. \] By condition (c), there exists some term whose sum is \(x_1 + x_{n-2}\). Since \(x_1 + x_{n-2} > x_{n-2},\) the only possibility is \(x_{n-1}\).

Next, observe that \(x_{n-3} < x_1 + x_{n-3} < x_1 + x_{n-2} = x_{n-1} < 2n\) so it follows that \(x_1 + x_{n-3} = x_{n-2}\). Repeating the logic \(n-3\) more times, we discover that the system of equations in the claim holds.

We now show that \(x_2 = 2x_1\). We have \[\begin{align*} &\quad x_1 + x_{n-2} = x_{n-1} \\ \iff&\quad x_1 + 2n - x_2 = 2n - x_1 \\ \iff&\quad x_2 = 2x_1. \end{align*}\]

Substituting the above into our equations and iteratively substituting thereafter, we find that \(x_2 = 2x_1\), \(x_3 = 3x_1\), \(\dots\), \(x_{n-2} = (n-2)x_1\), and \(x_{n-1} = (n-1)x_1\). Since \(2n - x_1 = x_{n-1} = (n-1)x_1\), we derive that \(x_1 = 2\). Therefore the only sequence is \(2, 4, 6, \dots, 2n-2\).

\(\blacksquare\)

Motivation

Once we derive the main claim that \(x_1 + x_{n-i} = x_{n-i+1}\) for each \(1 < i < n - 1\), the rest of the solution becomes details. The motivation for the key claim is to realize that the sum with the maximal gap between terms is extremely (no pun intended) well-defined. Then, it's easy to \(\textit{tip}\) the dominoe to get all the other equalities. This is a nice problem where analyzing extremities does wonders.
